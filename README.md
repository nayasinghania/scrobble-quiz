## Getting Started

### Prerequisites

- Python
- Node.js

### Setup

1. `npm install`
2. `cd src/app/api && pip install -r requirements.txt`

### Running

1. `npm run dev`
2. The frontend dev server will appear at `http://localhost:3000`
3. The backend dev server will appear at `http://127.0.0.1:8000`
